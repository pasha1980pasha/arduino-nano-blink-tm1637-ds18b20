# Arduino Nano — Коллекция примеров кода

## 📋 Общее описание

Данный репозиторий содержит **три готовых примера** для платформы **Arduino Nano** в среде разработки **PlatformIO**. Каждый пример представляет собой законченный скетч с подробным описанием подключения, необходимыми библиотеками и пояснениями по работе.

Примеры подходят как для начинающих (тест платы, работа с Serial), так и для более продвинутых проектов (внешние модули — 7-сегментный индикатор, цифровые датчики температуры).

---

## 📦 Содержание

| № | Название | Файл | Краткое описание |
|---|----------|------|------------------|
| 1 | **Blink** | `blink.ino` | Мигание встроенным светодиодом на D13 с выводом в Serial Monitor |
| 2 | **TM1637 Display** | `tm1637.ino` | Управление 4-разрядным 7-сегментным индикатором (счётчик, точки, форматирование) |
| 3 | **DS18B20 Scanner** | `ds18b20_scanner.ino` | Поиск адресов датчиков температуры DS18B20 на шине OneWire |

---

## 🔧 Общая конфигурация PlatformIO

Все примеры используют одинаковую базовую конфигурацию для Arduino Nano:

```ini
[env:arduino_nano]
platform = atmelavr
board = nanoatmega328
framework = arduino
board_bootloader = atmega328old
upload_speed = 57600
monitor_speed = 9600
upload_port = COM6
